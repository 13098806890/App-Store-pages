Support — Basketball Career

Welcome to Basketball Career (篮球生涯). If you encounter any issues, please use the following resources:

FAQ

1) Why are some buttons disabled?

Make sure you have:

  • Started a new game with the correct settings
  • Started the current period
  • The game is not paused or completed
  • Selected a player to record stats

2) Will my game data be lost?

The app auto-saves during gameplay. We also recommend manually saving to history and exporting backups regularly.

3) How to transfer data between devices?

Export your player/team/game data on the old device, then import on the new device using the share code or Bluetooth sync (Pro).

4) Can I recover deleted data?

Deleted data cannot be recovered. Please export backups before deletion.

5) Bluetooth sync is not working?

  • Make sure Bluetooth is enabled on both devices
  • Both devices should be within Bluetooth range
  • This is a Pro feature — subscribe to unlock

6) How to use AI summary?

  • Subscribe to Pro to unlock AI features
  • Go to Settings → AI, configure an API Key and test the connection
  • During game review, tap "AI Summary" to generate a summary

常见问题

1) 为什么有些按钮不可点击？

请先确认：

  • 已完成新比赛配置
  • 当前节次已开始
  • 比赛未暂停且未结束

2) 比赛数据会丢失吗？

应用支持比赛过程自动保存。建议在关键节点手动"存到历史"，并定期导出备份。

3) 更换手机后如何迁移数据？

在旧设备中导出球员/球队/比赛数据，再在新设备中导入（编码分享或蓝牙同步（Pro））。

4) 删除后能恢复吗？

通常不可恢复。请在删除前先导出备份。

5) 蓝牙同步无法使用？

  • 确保两台设备蓝牙已开启
  • 两台设备需在蓝牙范围内
  • 此为 Pro 功能，请订阅后使用

6) 如何使用 AI 总结？

  • 订阅 Pro 解锁 AI 功能
  • 在设置 → AI 中配置 API Key 并测试连接
  • 在比赛回顾中点击"AI 总结"生成总结

よくある質問

1) 一部のボタンが無効になるのはなぜですか？

以下をご確認ください：

  • 正しい設定で新しい試合を開始している
  • 現在のピリオドが開始している
  • 試合が一時停止または終了していない
  • 統計を記録する選手を選択している

2) 試合データは失われますか？

アプリは試合中に自動保存されます。定期的に手動保存し、バックアップをエクスポートすることをお勧めします。

3) デバイス間でデータを移行するには？

古いデバイスで選手/チーム/試合データをエクスポートし、新しいデバイスでコードまたはBluetooth同期（Pro）を使用してインポートします。

4) 削除したデータは復元できますか？

削除されたデータは復元できません。削除前にバックアップをエクスポートしてください。

5) Bluetooth同期が機能しません

  • 両方のデバイスでBluetoothが有効になっていることを確認
  • 両方のデバイスがBluetooth範囲内にあることを確認
  • これはPro機能です。購読すると使用可能になります

6) AI要約を使用するには？

  • Proに購読してAI機能をアンロック
  • 設定→AIでAPIキーを設定し、接続をテスト
  • 試合レビューで「AI要約」をタップ

지원 — Basketball Career

Basketball Career（篮球生涯）에 오신 것을 환영합니다. 문제가 발생하면 다음 리소스를 이용해 주세요.

자주 묻는 질문

1) 일부 버튼이 비활성화되는 이유는 무엇인가요?

다음을 확인하세요:

  • 올바른 설정으로 새 경기를 시작했는지
  • 현재 피리어드가 시작되었는지
  • 경기가 일시 중지되거나 종료되지 않았는지
  • 통계를 기록할 선수를 선택했는지

2) 경기 데이터가 손실되나요?

앱은 게임 중 자동 저장됩니다. 정기적으로 수동 저장하고 백업을 내보내는 것이 좋습니다.

3) 기기 간 데이터를 전송하려면?

이전 기기에서 선수/팀/경기 데이터를 내보내고, 새 기기에서 코드 또는 Bluetooth 동기화(Pro)를 사용하여 가져옵니다.

4) 삭제된 데이터를 복구할 수 있나요?

삭제된 데이터는 복구할 수 없습니다. 삭제 전에 백업을 내보내 주세요.

5) Bluetooth 동기화가 작동하지 않습니다

  • 두 기기 모두 Bluetooth가 켜져 있는지 확인
  • 두 기기가 Bluetooth 범위 내에 있는지 확인
  • Pro 기능입니다. 구독하면 사용할 수 있습니다

6) AI 요약을 사용하려면?

  • Pro를 구독하여 AI 기능 잠금 해제
  • 설정 → AI에서 API 키 구성 및 연결 테스트
  • 경기 검토에서 "AI 요약" 탭

Часто задаваемые вопросы

Убедитесь, что:

  • Вы начали новую игру с правильными настройками

  • Вы выбрали игрока для записи статистики

Приложение автоматически сохраняется во время игры. Рекомендуется регулярно сохранять вручную и экспортировать резервные копии.
