Privacy Policy

App: Basketball Career (篮球生涯)

1. What Information We Collect

Basketball Career does not collect personally identifiable information (such as name, phone number, ID number) by default.

Data entered by the user (e.g., teams, players, game records) is stored locally and used solely for the app's core functionality.

2. How Data Is Stored

  • Data is stored locally on your device (app sandbox).
  • The app does not automatically upload your game data to any server.
  • If you configure an API Key for AI summaries, the key is stored in iOS Keychain.
  • If you enable iCloud sync (Pro feature), selected game data will be stored in your personal iCloud account.

3. Permissions

The app may request the following system permissions:

  • Photo Library: Used only when you choose to set an avatar photo for a player.
  • Local Network: Used for Bluetooth sync to discover nearby devices (via Bonjour / Multipeer Connectivity).

  • The app does not serve advertisements or perform ad tracking.
  • The app does not sell your local game data to any third party.
  • When you use AI summary, only the current game data is sent to the AI provider (DeepSeek, OpenAI, or Anthropic) you configured. No other data is shared.
  • When you use import/export/share or Bluetooth sync, you actively choose the recipient and channel.

5. Children & Minors

This app does not target children for data collection. If a guardian believes a minor's data has been improperly processed, please contact us using the information below.

Basketball Career по умолчанию не собирает личную информацию, позволяющую идентифицировать пользователя (имя, номер телефона, удостоверение личности).

Данные, введённые пользователем (команды, игроки, записи матчей), хранятся локально и используются только для основных функций приложения.

2. Как хранятся данные

  • Данные хранятся локально на вашем устройстве (песочница приложения).
  • Приложение не загружает ваши игровые данные на сервер автоматически.
  • Если вы настраиваете ключ API для ИИ-резюме, ключ хранится в связке ключей iOS.
  • Если вы включаете синхронизацию iCloud (функция Pro), выбранные игровые данные будут храниться в вашей личной учётной записи iCloud.

3. Разрешения

Приложение может запрашивать следующие системные разрешения:

  • Фототека: Используется только при выборе фото аватара для игрока.
  • Локальная сеть: Используется для Bluetooth-синхронизации для обнаружения ближайших устройств.
  • Bluetooth: Используется для совместного подсчёта очков в реальном времени между устройствами.

Если вы откажете, соответствующие функции могут быть недоступны, но остальная часть приложения будет работать нормально.

4. Передача данных и третьи лица

  • Приложение не показывают рекламу и не отслеживает действия пользователей.

  • При использовании ИИ-резюме только данные текущего матча отправляются настроенному поставщику ИИ (DeepSeek, OpenAI или Anthropic).
