## 常见问题

**AI 法律顾问无法使用？**
需要在「设置」中配置 DeepSeek API Key。新用户可免费获取一定额度。

**如何获取 DeepSeek API Key？**
访问 [platform.deepseek.com](https://platform.deepseek.com)，注册账号后在「API Keys」页面创建密钥。

**API Key 安全吗？**
密钥加密存储在本机系统 Keychain 中（可选 iCloud 同步），不会上传至第三方服务器。

**法律数据需要联网吗？**
不需要。1500 余部法律法规内置于 App 中，完全离线可用。AI 咨询功能需要联网。

**数据多久更新一次？**
数据来源于全国人大法律法规数据库，随 App 版本更新。

**iCloud 同步不生效？**
请确认已登录 iCloud 账号，并在系统设置中开启「iCloud 云盘」。

**搜索找不到内容？**
全文搜索需输入至少 3 个字符，建议使用具体法律术语。可在设置中关闭「仅搜索标题」。
