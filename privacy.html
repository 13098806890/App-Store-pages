律疏隐私政策

最后更新：2026年5月

一、基本说明

律疏（以下简称"本应用"）是一款中国法律法规检索与咨询工具。我们高度重视用户隐私，本政策说明本应用收集、使用和存储哪些数据。

二、数据收集

本应用不收集任何个人信息。

  • 不需要注册账号，不收集姓名、手机号、邮箱
  • 不收集位置信息
  • 不收集设备标识符用于追踪
  • 不使用第三方广告 SDK
  • 不向任何服务器上传用户的使用数据或问答内容

三、本地存储数据

以下数据仅存储在您的设备本地或您的个人 iCloud 账户中，由您完全控制：

数据存储位置用途
上次阅读记录（法律 ID + 条文号）iCloud KV Store启动时恢复阅读位置
法律顾问对话历史iCloud Documents跨设备同步对话记录
App 偏好设置设备本地 UserDefaults记住用户的显示偏好
DeepSeek API Key设备本地 Keychain调用 AI 服务时使用

四、AI 法律顾问功能

使用 AI 法律顾问功能时：

  • 您需要提供自己的 DeepSeek API Key，该 Key 仅存储在您设备的 Keychain 中，不会上传至任何服务器
  • 您输入的问题和 AI 返回的回答直接在您的设备与 DeepSeek 服务器之间传输，本应用不经手、不存储、不分析这些内容
  • DeepSeek 的数据处理方式请参阅其隐私政策：deepseek.com/privacy

五、法律数据库

本应用内置的法律法规数据库随 App 安装包分发，完全离线可用，不涉及任何网络请求或数据上传。

六、iCloud 同步

若您的设备已登录 Apple ID 并开启 iCloud：

  • 阅读记录和对话历史会通过您的个人 iCloud 账户在您的设备间同步
  • 这些数据受 Apple 的隐私政策保护，本应用无法访问其他用户的 iCloud 数据

七、儿童隐私

本应用不针对 13 岁以下儿童，亦不故意收集儿童个人信息。

九、政策更新

如本隐私政策发生变更，我们将在 App 更新说明中告知。继续使用本应用即视为接受更新后的政策。
